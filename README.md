# NativeScript Vue + TailwindCSS

This project integrates [Vue](https://vuejs.org/) and [NativeScript](https://www.nativescript.org/), allowing you to build cross-platform iOS and Android apps using Vue.

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ ns run android
```
